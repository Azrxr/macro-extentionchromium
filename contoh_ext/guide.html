📘 Panduan Penggunaan Ghost Hands

Berikut adalah penjelasan setiap jenis aksi yang tersedia di tab Dirty Hands. Klik pada header untuk membuka/tutup panduan per bagian.

🧪 Sample Process – Instagram Simulator

Klik tombol di bawah untuk secara otomatis:

  • ✅ Membuka halaman simulasi Instagram
  • ✅ Mengisi tab Dirty Hands dengan skenario uji coba lengkap
  • ✅ Menjalankan semua step (Follow/Unfollow, kirim DM, scroll, dll)
🌐 Navigasi ke URL

Parameter: URL lengkap (https://...)

Contoh: https://example.com

Kegunaan: Memulai skenario di halaman tertentu.

🔗 Klik & Buka Tab Baru

Parameter: CSS selector elemen (biasanya link atau tombol).

Contoh: a, #openLink, .btn-new-tab

Kegunaan: Setelah klik, ekstensi akan mendeteksi tab baru yang terbuka, lalu beralih ke tab tersebut. Berguna untuk mengikuti link yang membuka halaman di tab baru.

❌ Tutup Tab Saat Ini

Parameter: Tidak perlu.

Kegunaan: Menutup tab aktif tempat ekstensi berjalan. Eksekusi akan berhenti karena side panel ikut tertutup. Cocok untuk membersihkan tab setelah skenario selesai.

🖱️ Klik elemen

Parameter: CSS selector (id, class, atribut, dll).

Dukungan indeks: Untuk memilih elemen ke-n dari banyak elemen dengan selector yang sama, tambahkan | indeks (1-based). Contoh: .btn | 2 akan mengklik tombol kedua dengan class "btn".

Contoh: #submitBtn, .btn-primary, button[type="submit"], .btn | 2

🖱️ Klik berdasarkan teks (case-sensitive)

Parameter: Teks yang ingin dicari.

Dukungan wildcard: *teks* (mengandung), teks* (diawali), *teks (diakhiri).

Indeks: "Simpan | 2" untuk yang kedua, "Simpan | last" untuk terakhir.

Double click: Isi nilai tambahan (extra) dengan "double" untuk melakukan double click.

Contoh: *hapus*, Simpan*, *Simpan, Login | 1, "Hapus | 2" dengan extra "double"

🖱️ Klik semua elemen (satu per satu)

Parameter: CSS selector yang menunjuk ke beberapa elemen.

Nilai tambahan (opsional): delay antar klik dalam ms (default 500).

Contoh: .btn-delete, .item

🖱️ Hover / Mouse Over

Parameter: CSS selector.

Contoh: .menu-item, #profilePic

⌨️ Ketik teks

Parameter: CSS selector target.

Nilai tambahan: teks yang akan diketik.

Contoh: selector: #username, teks: admin

Fitur: Gunakan {{line}} untuk mengambil data dari dataset baris per baris.

Mendukung contenteditable – otomatis mendeteksi dan mengetik di editor kaya (seperti WhatsApp web, Google Docs).

📤 Upload File

Parameter: CSS selector untuk input file.

Nilai tambahan: Path ke file (tidak bisa otomatis, akan membuka dialog).

Contoh: selector: #fileInput, nilai: /path/to/file.txt

📋 Pilih Dropdown (by text/value)

Parameter: Format selector|teks atau selector|value:valueText.

Contoh: #country|Indonesia, #provinsi|value:JB

⌨️ Tekan Tombol Keyboard

Parameter: Kombinasi tombol (ENTER, TAB, CTRL+C, SHIFT+TAB, dll). Lihat daftar di fitur Suggest.

Nilai tambahan (opsional): selector target (kosongkan untuk menggunakan elemen yang sedang aktif).

Contoh: ENTER (kirim form), CTRL+ENTER (kirim pesan chat), TAB (pindah field).

📜 Scroll halaman / elemen

Parameter: Angka (pixel) atau format selector|arah|jarak.

Contoh: 300 (turun), -200 (naik), #scrollArea|down|200, #target|intoView.

⏳ Tunggu element muncul

Parameter: CSS selector.

Juga mendukung "not exists": not exists .loading-spinner untuk menunggu elemen hilang.

Timeout: 30 detik.

⏳ Tunggu Hingga (Wait Until) – polling kondisi

Fungsi: Menunggu hingga kondisi tertentu terpenuhi, dengan polling setiap 0.5 detik. Cocok untuk menunggu konten dinamis yang tidak bisa di-handle oleh waitForElement biasa (misal menunggu teks berubah, variabel mencapai nilai).

Parameter: Format kondisi sama dengan if_start (exists, not exists, text, variable).

Nilai tambahan (opsional): timeout dalam milidetik (default 30000).

Contoh: exists .loading-spinner | 10000 (tunggu hingga spinner loading muncul selama 10 detik). Atau text #status = "Selesai".

Dukungan indeks: exists .item | 2 (tunggu elemen ke-2 dengan class item muncul).

⏳ Tunggu halaman selesai loading

Parameter: Tidak perlu.

⏳ Tunggu Network Idle

Parameter: Timeout (ms).

Nilai tambahan (opsional): Durasi idle (ms), default 500.

⏱️ Delay / jeda

Parameter: durasi dalam milidetik. Bisa juga range 500-1000 untuk random.

Contoh: 1000 = 1 detik, 500-1500 = delay random antara 500-1500ms.

🔁 Start Loop

Parameter: jumlah perulangan (angka) atau "auto" (sesuai jumlah baris dataset).

Catatan: Harus diakhiri dengan End Loop.

Di dalam loop, gunakan {{line}} untuk mengambil baris dataset saat ini, dan {{line[0]}} jika dataset menggunakan separator ;.

🔚 End Loop

Parameter: Tidak perlu.

⏹️ Break (keluar dari loop)

Fungsi: Menghentikan perulangan meskipun belum mencapai batas.

Parameter: Tidak perlu.

🔀 If / Else If / Else (Percabangan)

Parameter (If / Else If): Kondisi (exists, not exists, text, variable).

Parameter (Else): kosong.

Contoh kondisi:

  • exists #selector – elemen ada
  • not exists #selector – elemen tidak ada
  • text #selector = "value" – teks sama persis
  • text #selector contains "value" – teks mengandung
  • variable nama = "nilai" – bandingkan variabel (string)
  • variable nama > 10 – bandingkan numerik
🔚 End If

Parameter: Tidak perlu.

🔍 Ambil Teks/Atribut ke Variabel

Parameter: Format selector|atribut (text, value, html, atau nama atribut).

Nilai tambahan: nama variabel.

Contoh: #total|text simpan ke "harga", lalu gunakan {{harga}} di step lain.

⚡ Execute JavaScript

Parameter: Kode JavaScript.

Nilai tambahan (opsional): nama variabel untuk menyimpan hasil return dari script.

Contoh: document.title disimpan ke variabel "judul".

🔄 Retry

Parameter: jumlah percobaan (angka).

Kegunaan: Jika step berikutnya gagal, akan diulang sebanyak N kali sebelum error.

📊 Collect Data

Fungsi: Mengambil data terstruktur dari halaman.

Parameter (opsional): Format itemSelector || kolom1:selector1|atribut, kolom2:selector2|text. Jika kosong, akan menggunakan konfigurasi dari tab Data Collector.

Contoh: tr.product || nama:.title|text, harga:.price|text

Batasan baris: Tergantung paket lisensi (gratis 100 baris, premium lebih banyak).

🔔 Tampilkan Alert (Bubble Ghost)

Parameter: Pesan teks atau CSS selector (diawali #, ., [, dll) untuk mengambil teks dari elemen halaman.

Nilai tambahan: format tipe|durasi|posisi (default info|3000|center).

Placeholder: Gunakan {{selector}} di dalam pesan untuk menyisipkan teks dari elemen. Contoh: Halo, {{#userName}}! Selamat datang.

Contoh: #statusMsg (ambil teks dari elemen), Proses selesai (teks biasa), Pesan: {{.message}} (gabungan).

🎥 Rekam Aktivitas

Fungsi: Merekam klik, input teks, dan dropdown menjadi step otomatis.

Cara: Klik tombol "🔴 Rekam Aktivitas", lakukan aksi di halaman, klik "⏹️ Hentikan Rekam".

🎯 Tips Memilih Selector yang Tepat
  • Gunakan ID jika ada → #submitBtn
  • Gunakan atribut unik → [data-id="123"]
  • Untuk elemen dengan class dinamis: [class^="prefix-"] (diawali) atau [class*="teks"] (mengandung)
  • Hindari selector yang terlalu panjang (hasil rekaman). Gunakan Suggest untuk meringkas.
  • Untuk elemen di dalam shadow DOM, ekstensi sudah mendukung, cukup gunakan selector seperti biasa.
📊 Tunggu Perubahan Nilai (Wait for Value Change)

Fungsi: Menunggu hingga nilai dari suatu elemen (teks, value, atau atribut) berubah sesuai kondisi. Berguna untuk mendeteksi pesan baru, harga berubah, stok berubah, dll.

Parameter format: selector|atribut|kondisi|threshold(opsional)

  • selector – CSS selector elemen (mendukung indeks: selector|2)
  • atribut – text, value, html, atau nama atribut (default text)
  • kondisi – increase (nilai naik), decrease (turun), change (berubah), greater_than (naik lebih dari threshold)
  • threshold (opsional) – angka minimal perubahan untuk kondisi greater_than

Nilai tambahan: timeout dalam milidetik (default 30000).

Contoh:

  • #badge|text|increase – tunggu angka pada badge naik
  • .price|text|greater_than|5000 – tunggu harga naik lebih dari 5000
  • [data-testid="list-item-0"] [data-testid="icon-unread-count"]|text|change – tunggu badge pesan berubah
  • .status|text|decrease – tunggu teks status berkurang (misal angka stok)

Dukungan indeks: .item|2|text|increase – tunggu elemen ke-2 dengan class item

Placeholder: Bisa gunakan {{variabel}} di selector dan timeout.

🖥️ Tampilkan Overlay Penuh (Fullscreen)

Parameter: Teks atau CSS selector (diawali #, ., [, >) untuk mengambil teks dari elemen halaman. Bisa juga gunakan placeholder {{selector}} untuk menyisipkan teks.

Nilai tambahan: format tipe|durasi (tipe: info, success, warning, error; durasi dalam ms, 0 untuk tidak auto close).

Contoh: Proses selesai | success|3000 atau #statusMsg | info|5000.

Fungsi: Menampilkan overlay fullscreen dengan latar belakang gelap, pesan di tengah, dan tombol tutup. Overlay akan hilang otomatis setelah durasi (jika durasi > 0). Cocok untuk pemberitahuan penting yang tidak boleh terlewat.

🧩 Skenario Kompleks (Loop + If + Break + NetworkIdle)

Tujuan: Memproses daftar item, mengedit yang memenuhi syarat, berhenti jika ditemukan error.

1. Navigasi ke URL (https://example.com/list)
  2. Start Loop (auto dengan dataset berisi indeks 1..10)
  3.    WaitForNetworkIdle (5000)
  4.    Klik element (tr:nth-child({{line}}) .edit-btn)
  5.    WaitForElement (.modal)
  6.    Get Variable (.points|text) simpan "poin"
  7.    If (variable poin > 100)
  8.        SelectDropdown (#status|Approved)
  9.        Klik (.save-btn)
  10.   Else
  11.       Klik (.cancel-btn)
  12.   End If
  13.   WaitForPageLoad
  14.   If (exists #errorMessage)
  15.       Break
  16.   End If
  17. End Loop
  18. Close Tab
🎲 Random Delay (jeda acak)

Parameter: Rentang waktu dalam milidetik, ditulis min-max. Contoh: 500-1500 akan menghasilkan delay antara 500 ms hingga 1500 ms secara acak.

Nilai tambahan: Tidak digunakan (kosongkan).

Contoh: 1000-3000 untuk jeda acak 1-3 detik.

Dukungan placeholder: Bisa menggunakan {{variabel}} untuk menentukan range, misal {{minDelay}}-{{maxDelay}}.

📝 Set Variabel (deklarasi nilai)

Fungsi: Membuat variabel dengan nilai tertentu (string, angka, array, object) tanpa mengambil dari halaman.

Parameter: Nilai yang akan disimpan. Bisa berupa:

  • Teks biasa: Hello World
  • Angka: 123 atau 3.14
  • Array JSON: ["a","b","c"]
  • Object JSON: {"nama":"John","umur":30}
  • Placeholder: {{variabelLain}}

Nilai tambahan: Nama variabel (wajib).

Contoh: Parameter ["item1","item2"], extra daftar → variabel daftar berisi array.
